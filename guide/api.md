# API

Monibuca 提供 HTTP 形式的 API 调用，方便用户对 Monibuca 实例进行信息的读取和指令下达。

Monibuca API 分为系统 API 和 插件 API 。

## 系统 API

系统 api 在 apifox 上维护，如下图所示：
